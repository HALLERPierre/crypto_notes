# Installation
## Python 3
```shell
pip install PyCrypto
pip install flask
pip install flask-cors
```
You will also need to have `python-dev` installed
## Node
```shell
npm install
```
# Execution
* Run flask server :
```shell
python crypto/app.py
```
* Run javascript bundler :
```shell
npm run dev
```
